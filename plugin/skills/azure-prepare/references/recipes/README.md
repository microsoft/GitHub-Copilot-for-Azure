# Recipes

Infrastructure-as-Code recipes for Azure preparation.

| Recipe | When to Use |
|--------|-------------|
| [AZD](azd/README.md) | Default. New projects, multi-service apps, want `azd up` |
| [AZCLI](azcli/README.md) | Existing az scripts, imperative control, custom pipelines |
| [Bicep](bicep/README.md) | IaC-first, no CLI wrapper, direct ARM deployment |
| [Terraform](terraform/README.md) | Multi-cloud, existing TF expertise, state management |

See [Selection Guide](../recipe-selection.md) for detailed criteria.
